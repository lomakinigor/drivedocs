# Язык интерфейса

Всё, что видит пользователь — на русском: UI-тексты, подписи, подсказки, ошибки, empty states, статусы, отчёты, PDF, уведомления, онбординг.

**Допустимые исключения:** аббревиатуры (ИП, ООО, ИНН, VIN, PDF, API, OCR, ID), названия технологий (Supabase, Stripe, TypeScript), спецсимволы, коды, e-mail, URL.

Если внешняя система возвращает текст на английском — не показывать пользователю напрямую. Отображать понятный русский текст, оригинал — только в логи.

**Проверка перед завершением задачи:** нет ли новых строк на английском в UI и документах.
