## 🎯 Топ-5 что чинить сейчас

### 1. Закрыть Visual Phase — TripsPage, sheets, OnboardingWizard (1.2) — критично

**Не так:** Старые экраны на синем Tailwind мешаются с новыми oklch-indigo (FeedbackSheet, F-032 toggle, EssentialsReminder). Чем больше нового, тем сильнее впечатление «сшито из лоскутов».

**Что делать:** один PR на страницу. TripsPage → QuickTripSheet → TripDetailSheet → QuickReceiptSheet → OnboardingWizard. Каждый по шаблону HomePage/Settings: oklch indigo, Sora, slate-500 (не 400), радиусы 18px.

**Эффект:** 1.2 с 1 → 3, 1.4 с 2 → 3, 1.8 с 2 → 3. Сразу +4 балла, выводит дизайн с 67 → 83%.

### 2. Прозрачность цен на Welcome (3.3, 3.9, 3.1)

**Не так:** Welcome молчит о платности. «До 180 000 ₽/год» третьим буллетом. Pro обнаруживается только в Settings — это сюрприз, граничит с обманом ожиданий.

**Что делать:**
- Поднять «налоговый вычет 180 000 ₽» первым буллетом (формула «Возвращайте до …»)
- Под главной CTA строкой `text-xs text-slate-500`: «Бесплатно в MVP • Цены появятся позже • Отмена в один тап»
- Добавить четвёртый буллет «И мы быстро отвечаем — пишите в @drivedocs_bot» (закрывает 3.8 + социальное доказательство присутствия команды)

**Эффект:** 3.1 1→2, 3.3 1→3, 3.9 2→3. +4 балла, выводит выгоду с 59 → 74% — уже у порога.

### 3. HelpInfoSheet на оставшиеся жаргонные точки (2.3, 2.6)

**Не закрыто:** `TaxModeStep`, `ReceiptCategory`, поле «Цель поездки», расшифровка «АМ-23-р» рядом с блоком ГСМ.

**Что делать:** добавить `HELP_TAX_MODE`, `HELP_RECEIPT_CATEGORIES`, `HELP_TRIP_PURPOSE`, привязать `HelpFuelNorms` к ярлыку блока ГСМ в путевом. Это уже готовый паттерн (F-021).

**Эффект:** 2.3 1→3, 2.6 2→3. +3 балла, удобство 70 → 80% — на пороге.

### 4. Сформулировать экономию времени в UI (3.5)

**Не так:** F-032 расширенный бланк с расчётом ГСМ = ~20 минут на каждый путевой. Но это **не сказано** — пользователь не понимает что мы для него делаем.

**Что делать:** в `WaybillPreviewSheet` под toggle добавить строчку:

> «Расширенный бланк экономит ~20 минут — расчёт нормы ГСМ по АМ-23-р делается за вас»

И на ReportsPage hero-карте — «Расчёт ГСМ автоматический, не считайте вручную».

**Эффект:** 3.5 2→3.

### 5. Контраст slate-400 → slate-500 на оставшихся экранах (1.7)

**Не так:** SettingsPage переведена, остальные нет. KPI-лейблы на HomePage, чек-категории, sub-text по-прежнему 3.3:1.

**Что делать:** автоматизированная замена (grep + sed) `text-slate-400` → `text-slate-500` для всего, что не иконка. Глобальный focus-ring в `index.css`.

**Эффект:** 1.7 2→3. +1 балл.
